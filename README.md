📊 NIFTY Pre-Market Intelligence Dashboard

🚀 A real-time AI-driven market intelligence system that transforms global news into actionable sector insights for NIFTY & BANKNIFTY.

🔥 Project Overview
Most dashboards show what happened.
This tool focuses on what is likely to happen next.

It collects live news from multiple sources, analyzes sentiment, maps it to sectors, and provides:
👉 Market Verdict (Bullish / Bearish / Neutral)
👉 Sector-wise Opportunity Heatmap
👉 India-specific macro impact analysis
👉 Trading bias for NIFTY & BANKNIFTY

⚙️ Key Features
✅ Live News Aggregation (No API Required)

Uses RSS feeds (Reuters, Economic Times, Moneycontrol, Google News)
No API keys, no login required

✅ AI-Based Sentiment Engine
Classifies news into Bullish / Bearish / Neutral
Uses keyword-based + macro-rule logic

✅ Sector Intelligence Mapping
Maps news → sectors like IT, Banking, Auto, FMCG, Energy
Generates a sector heatmap with trading signals

✅ India Market Impact Engine 🇮🇳
Converts global events into India-specific outcomes
Example:
Crude ↑ → FMCG ↓, Auto ↓
Rate Cut → Banking ↑, Realty ↑

✅ Trading Verdict System
Bullish probability score (%)
Actionable bias:
Buy on dips
Sell on rise
Wait for breakout


🧠 How It Works
Fetches live market + geopolitical news
Cleans and filters headlines
Applies sentiment scoring
Maps events to sectors using predefined rules
Generates:
Sector scores
Market sentiment
Trading bias
